SHARAD (Shallow Radar) data is freely available at:
https://pds-geosciences.wustl.edu/missions/mro/sharad.htm

This example data has been converted to SEG-Y format, and
uses the open source package segyio to handle the data.
Data hosted on the PDS will be in IMG format.
To work with data in IMG format, consider using the 
Planetary Data Reader (pdr) package: 
https://pdr.readthedocs.io/en/latest/
